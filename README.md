##  Run demo.m to perform simulation.

Use these variables to control simulation. 
1. totalAnimals
2. simulationLength
3. powerDifference
4. pastureRate
5. deathRate
6. shelterRate
7. starvationRate
8. fieldOfView
9. numberOfEnemies
